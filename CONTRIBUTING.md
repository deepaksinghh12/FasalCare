# Contributing to Project Kisan

Thank you for your interest in contributing to Project Kisan! We welcome contributions from the community to help us improve this platform for farmers.

## Getting Started

1.  **Fork** the repository.
2.  **Clone** your fork locally.
3.  Create a new **branch** for your feature or bug fix:
    ```bash
    git checkout -b feature/my-new-feature
    ```

## Development Workflow

-   **Install Dependencies**: `npm install`
-   **Run Dev Server**: `npm run dev`
-   **Database**: Ensure you have `MONGODB_URI` set in `.env.local`.

## Pull Request Process

1.  Ensure your code follows the existing style (Project uses Prettier/ESLint).
2.  Update `README.md` if you change any functionality.
3.  Submit a Pull Request to the `main` branch.

## Code of Conduct

Please be respectful and professional in your interactions. We are here to learn and build together.
